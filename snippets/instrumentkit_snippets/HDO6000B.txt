Sure! Here's an example script that uses Instrumentkit to connect to a HDO6000B Oscilloscope:

```python
import instrumentkit as ik

# Connect to the oscilloscope
address = "TCPIP0::192.168.0.10::INSTR"
oscilloscope = ik.teledyne.MAUI.open_visa(address)

# Print the oscilloscope information
print(oscilloscope.show_id())

# Set up the first channel
channel = oscilloscope.channel[0]
channel.trace = True  # Turn on the trace
channel.coupling = channel.Coupling.dc50  # Set the coupling to 50 Ohm
channel.scale = 1  # Set the vertical scale to 1V/division

# Set the timebase
oscilloscope.time_div = 200e-9  # Set the time per division to 200 ns

# Start the trigger in automatic mode
oscilloscope.run()

# Read and plot the waveform from the first channel
xdat, ydat = channel.read_waveform()
import matplotlib.pyplot as plt
plt.plot(xdat, ydat)
plt.xlabel('Time (s)')
plt.ylabel('Voltage (V)')
plt.show()

# Disconnect from the oscilloscope
oscilloscope.close()
```

Make sure to replace the `address` variable with the actual IP address or VISA resource string of your oscilloscope.
