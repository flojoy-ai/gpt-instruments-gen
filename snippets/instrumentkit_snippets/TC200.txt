Here is an example Python script that uses Instrumentkit to connect to a TC200 Temperature Controller:

```python
from instrumentkit import Instrument, ConnectionType

# Define the TC200 Temperature Controller class
class TC200(Instrument):
    def __init__(self, connection_type, address):
        super().__init__(connection_type, address)

    def get_temperature(self):
        return self.query("READ:TEMP?")

# Create an instance of the TC200 Temperature Controller
tc200 = TC200(ConnectionType.SERIAL, "/dev/ttyUSB0")

# Connect to the TC200 Temperature Controller
tc200.connect()

# Get the temperature from the TC200 Temperature Controller
temperature = tc200.get_temperature()
print(f"Temperature: {temperature}")

# Disconnect from the TC200 Temperature Controller
tc200.disconnect()
```

Note: This code assumes that you have already installed the `instrumentkit` package.
