To connect to a FSW-0200 using Instrumentkit, you can follow the steps below:

1. Install the Instrumentkit library by running `pip install instrumentkit` in your terminal.

2. Import the necessary modules from Instrumentkit:

```python
from instrumentkit import FSW0200
from instrumentkit import ConnectionType
```

3. Create an instance of the FSW0200 class and specify the connection type:

```python
fsw = FSW0200(connection_type=ConnectionType.VISA)
```

4. Connect to the FSW-0200 instrument using the appropriate address:

```python
fsw.connect('TCPIP::192.168.1.1::INSTR')
```

Replace `'TCPIP::192.168.1.1::INSTR'` with the actual IP address or VISA resource string of your FSW-0200 instrument.

5. Now you can use the `fsw` object to send commands and receive responses from the instrument. For example, you can query the instrument's identification:

```python
idn = fsw.query('*IDN?')
print('Instrument ID:', idn)
```

6. Finally, don't forget to disconnect from the instrument when you're done:

```python
fsw.disconnect()
```

Here's the complete script:

```python
from instrumentkit import FSW0200
from instrumentkit import ConnectionType

# Create an instance of the FSW0200 class
fsw = FSW0200(connection_type=ConnectionType.VISA)

# Connect to the FSW-0200 instrument
fsw.connect('TCPIP::192.168.1.1::INSTR')

# Query the instrument's identification
idn = fsw.query('*IDN?')
print('Instrument ID:', idn)

# Disconnect from the instrument
fsw.disconnect()
```

Make sure to replace `'TCPIP::192.168.1.1::INSTR'` with the actual IP address or VISA resource string of your FSW-0200 instrument.
