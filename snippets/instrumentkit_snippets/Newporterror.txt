To connect to a Newport Agilis Controller using Instrumentkit, you can use the following Python script:

```python
import instrumentkit as ik

# Create a connection to the Newport Agilis Controller
agl = ik.newport.AGUC2.open_serial(port='COM5', baud=921600)

# Reset the controller
agl.reset_controller()

# Print the firmware version
print(agl.firmware_version)

# Move the X axis relative by 1000 steps
agl.axis["X"].move_relative = 1000

# Activate jogging in mode 3 for the X axis
agl.axis["X"].jog = 3

# Stop the X axis
agl.axis["X"].stop()

# Query the step amplitude for the X axis
step_amplitude = agl.axis["X"].step_amplitude
print(f"Step amplitude for X axis: {step_amplitude}")

# Set the step amplitude for the X axis to +10 and -20
agl.axis["X"].step_amplitude = [10, -20]

# Close the connection to the Newport Agilis Controller
agl.close()
```

Make sure to replace `'COM5'` with the correct serial port for your Newport Agilis Controller.
