To connect to a Newportesp 301 Positional Controller using Instrumentkit, you can use the following Python script:

```python
import instrumentkit as ik

# Connect to the Newportesp 301 Positional Controller
controller = ik.newport.NewportESP301.open_tcp(ip_address='192.168.1.1', port=50000)

# Set the velocity of the X axis to 1000 steps/s
controller.axis['X'].velocity = 1000

# Move the X axis to position 5000 steps
controller.axis['X'].move_absolute(5000)

# Wait until the X axis reaches the target position
while not controller.axis['X'].is_stopped:
    pass

# Print the current position of the X axis
print(f"Current position of X axis: {controller.axis['X'].position} steps")

# Disconnect from the controller
controller.close()
```

Make sure to replace `'192.168.1.1'` with the actual IP address of your Newportesp 301 Positional Controller. Also, adjust the axis name (`'X'`) and the target position (`5000`) according to your setup.
