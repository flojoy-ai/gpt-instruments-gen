Here is a Python script that uses Qcodes Community to connect to a LDA-906V-8 Digital Attenuator:

```python
from qcodes import Station
from qcodes.instrument_drivers.vaunix.LDA import LDA

# Create a station to hold the instrument
station = Station()

# Connect to the LDA-906V-8 Digital Attenuator
attenuator = LDA('attenuator', serial_number=906, channel_names={1: 'ch1', 2: 'ch2', 3: 'ch3', 4: 'ch4', 5: 'ch5', 6: 'ch6', 7: 'ch7', 8: 'ch8'})
station.add_component(attenuator)

# Print the instrument information
print(attenuator.get_idn())

# Set the attenuation of channel 1 to 10 dB
attenuator.ch1.attenuation(10)

# Get the attenuation of channel 1
print(attenuator.ch1.attenuation())

# Close the connection to the instrument
attenuator.close()
```

Note: Make sure you have the necessary DLL files (`VNX_atten64.dll` or `VNX_atten.dll`) in the correct directory and update the `dll_path` in the script if necessary.
