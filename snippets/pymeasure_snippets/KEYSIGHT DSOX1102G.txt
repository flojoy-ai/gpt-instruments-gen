Sure! Here's a Python script that uses Pymeasure to connect to a KEYSIGHT DSOX1102G Oscilloscope:

```python
from pymeasure.adapters import VISAAdapter
from pymeasure.instruments.keysight import KeysightDSOX1102G

# Connect to the oscilloscope
adapter = VISAAdapter("USB0::0x0957::0x179B::MY44012345::INSTR")
oscilloscope = KeysightDSOX1102G(adapter)

# Perform some operations on the oscilloscope
oscilloscope.clear_status()
oscilloscope.factory_reset()
oscilloscope.default_setup()

# Set up channel 1
oscilloscope.ch1.setup(coupling="ac", range=0.1, scale=0.01)

# Set up timebase
oscilloscope.timebase_setup(mode="main", offset=0, horizontal_range=0.1, scale=0.001)

# Acquire waveform data from channel 1
oscilloscope.digitize("channel1")
data, preamble = oscilloscope.download_data(source="channel1", points=1000)

# Print the acquired data and waveform preamble
print("Data:", data)
print("Preamble:", preamble)

# Disconnect from the oscilloscope
oscilloscope.shutdown()
```

Note: Make sure to replace the VISA address in the `VISAAdapter` constructor with the appropriate address for your oscilloscope.
