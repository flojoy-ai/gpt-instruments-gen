```python
import PyTango

# Create a DeviceProxy object to connect to the Keithley 3706 Multimeter
device_proxy = PyTango.DeviceProxy("device/hostname/Keithley3706")

# Read the voltage measurement from channel 1
voltage = device_proxy.read_attribute("Voltage1").value

# Print the voltage measurement
print("Voltage: ", voltage)
```

Explanation:
1. Import the `PyTango` module.
2. Create a `DeviceProxy` object by passing the device name as a string to the `DeviceProxy` constructor. Replace `"device/hostname/Keithley3706"` with the actual device name or address.
3. Use the `read_attribute` method of the `DeviceProxy` object to read the voltage measurement from channel 1. Replace `"Voltage1"` with the actual attribute name for voltage measurement.
4. Access the `value` attribute of the returned result to get the voltage measurement.
5. Print the voltage measurement.
