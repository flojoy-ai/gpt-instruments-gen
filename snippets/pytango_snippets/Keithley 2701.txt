```python
import PyTango

# Create a DeviceProxy object to connect to the Keithley 2701 Multimeter
device_proxy = PyTango.DeviceProxy("device/hostname/Keithley2701/1")

# Read the voltage measurement from the multimeter
voltage = device_proxy.read_attribute("Voltage").value

# Print the voltage measurement
print("Voltage: ", voltage)
```

Explanation:
1. Import the `PyTango` module.
2. Create a `DeviceProxy` object named `device_proxy` to connect to the Keithley 2701 Multimeter. Replace `"device/hostname/Keithley2701/1"` with the actual Tango device name of your Keithley 2701 Multimeter.
3. Use the `read_attribute` method of the `device_proxy` to read the voltage measurement from the multimeter. Replace `"Voltage"` with the actual attribute name of the voltage measurement on your Keithley 2701 Multimeter.
4. Access the `value` attribute of the returned value to get the voltage measurement.
5. Print the voltage measurement.
