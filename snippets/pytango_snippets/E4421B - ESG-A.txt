```python
import pytango

# Create a Tango DeviceProxy for the E4421B RF Signal Generator
device_proxy = pytango.DeviceProxy("device_name")

# Connect to the device
device_proxy.connect()

# Set the frequency range
device_proxy.frequency_low = 250e3
device_proxy.frequency_high = 3e9
```

Explanation:
1. Import the `pytango` module.
2. Create a `DeviceProxy` object for the E4421B RF Signal Generator by providing the device name as an argument to the `DeviceProxy` constructor. Replace `"device_name"` with the actual name of the device.
3. Call the `connect()` method on the `DeviceProxy` object to establish a connection to the device.
4. Set the frequency range of the RF Signal Generator by assigning the desired values to the `frequency_low` and `frequency_high` attributes of the `DeviceProxy` object.
