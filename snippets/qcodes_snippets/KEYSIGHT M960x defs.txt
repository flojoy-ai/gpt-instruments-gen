Sure! Here's an example script that uses Qcodes to connect to a KEYSIGHT M960x Power Meter:

```python
import qcodes as qc
from qcodes.instrument_drivers.Keysight.Keysight_M960x import Keysight_M960x

# Connect to the power meter
power_meter = Keysight_M960x("power_meter", "TCPIP0::192.168.1.1::inst0::INSTR")

# Print the power meter identification
print("Power Meter Identification:")
print("Vendor:", power_meter.vendor())
print("Model:", power_meter.model())
print("Serial Number:", power_meter.serial_number())

# Set the power meter to measure power
power_meter.measurement_function("POWER")

# Set the power meter range
power_meter.range_auto(True)

# Read the power measurement
power = power_meter.power()

# Print the power measurement
print("Power:", power, "dBm")

# Close the connection to the power meter
power_meter.close()
```

Note: Replace `"TCPIP0::192.168.1.1::inst0::INSTR"` with the actual VISA address of your KEYSIGHT M960x Power Meter.

This script connects to the power meter, retrieves its identification information, sets the measurement function to power, enables auto-ranging, reads the power measurement, and then closes the connection to the power meter.
